# JobsWrapper
This is an example project with a simple api to show off automated REST testing in Kotlin. Using:
* Production framework
  * Ktor
  * Kotlinx
  * Maven
* Test Framework 
  * Kotlin-test + JUnit5
  * RESTAssured or just use ktor + assertJ?
  * Spek for BDD
